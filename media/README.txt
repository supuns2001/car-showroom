MEDIA FOLDER
============
Add multimedia testimonial files here.

- testimonial.mp4          : Customer video testimonial
- audio-testimonial.mp3    : Customer audio testimonial

In reviews.html, replace the placeholder divs with:
<video src="media/testimonial.mp4" controls width="100%"></video>
<audio src="media/audio-testimonial.mp3" controls style="width:100%"></audio>
